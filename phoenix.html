Phoenix - Duelist

kekuatan bintang Phoenix terpampang dalam gaya bertarungnya, mengobarkan medan perang dengan kilat dan api. Saat sendiri maupun dibantu rekannya, dia akan menerjang ke pertarungan sesuka hatinya.

Kemampuan Phoenix

  • Blaze:membuat dinding api yang menghalangi pandangan dan mencederai pemain yang melewatinya.
  • Curveball: Melemparkan bola api yang akan membutakan lawan.
  • Hot hands: Melemparkan bola api yang akan melukai lawan dan bisa menyembuhkan pengguna.
  • Run it back: Saat kemampuan ini aktif, mati atau membiarkan pewaktu habis akan mengakhiri kemampuan ini dan mengembalikan Phoenix ke lokasi ini dengan health penuh.
